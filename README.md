# SVM Classifier Project

## Objective
To implement and analyze SVM on a real-world dataset.

## Dataset
Breast Cancer dataset (from sklearn)

## Methods Used
- SVM classifier
- Kernels: Linear, RBF, Polynomial
- Hyperparameter tuning (GridSearchCV)

## Result
RBF kernel gave the best accuracy.

## Conclusion
SVM performs well for classification problems with proper kernel selection.
